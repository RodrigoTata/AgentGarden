# Multiagent Garden

## Overview

A sandbox repository that hosts a collection of agent workflows used to orchestrate various development and security tasks across the broader MeedTrack and TataDeli projects. The repository contains reusable **workflow SKILL.md** files that define step‑by‑step processes for automated agents.

## Features
- **Cyber‑audit** workflow for full‑stack security assessment of NestJS/Angular/AI components.
- **Empacar** workflow to package changes and generate documentation.
- Additional utilities: context compactor, TDD scaffolding, issue generation, PRD creation, etc.

## Tech Stack
- **Node.js** (TypeScript) – for workflow definitions and scripts.
- **Git** – version control.
- **Markdown** – documentation and workflow specifications.

## Workflows
- **/cyber-audit** – Performs static security analysis and generates a `security_audit_report.md`.
- **/empacar** – Synthesizes recent changes into human‑readable documentation and prepares a clean commit.
